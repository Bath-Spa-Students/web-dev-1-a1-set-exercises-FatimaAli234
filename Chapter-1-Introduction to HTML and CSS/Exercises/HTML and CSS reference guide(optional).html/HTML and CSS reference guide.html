✦✦✦HTML and CSS Reference guide✦✦✦

This is an HTML and CSS quick-reference guide which covers all the tags we've covered so far and (few extras).

☆HTML Headings☆

  1. Headings in HTML, denoted by <h1> to <h6> tags, establish the hierarchy of the content.
  2. <h1> represents the most important heading, while <h6> is used for subheadings.

    
Heading 1

    <h1> This is heading 1</h1>
    
Heading 2

    <h2> This is heading 2</h2>
    
Heading 3

    <h3> This is heading 3</h3>
    
Heading 4

    <h4> This is heading 4</h4>
    
Heading 5

    <h5> This is heading 5</h5>
    
Heading 6

    <h6>This is heading 6</h6>

☆Paragraph☆

2.A paragraph in HTML is a block-level element used to organize and display text content.

<p>Did you know the average spends around 6 hours a day online?</p>

☆Horizontal line☆

3.The <hr>tag is used to create a horizontal line,or horizontal rule,to visually separate content within the HTML document.

<hr>

☆Strong Emphasis☆

4.The <Strong>tag is used to indicate the text with strong importance or emphasis.

<p>This is very <strong> Important</strong></p>

☆Italic Text☆

5.The <i>tag is used to render text in Italics.

<p>This is <i>italicized</i> text</p>.

☆Mark Tag☆

6.The <m> tag is used to highlight parts of the text.

<p>This is <mark>highlighted</mark> text.</p>

☆Abbr Tag

7.Defines an Abbrevation or acronym,providing an explanation when hovered over.

<p><abbr>HTML</abbr>is a fundamental language for creating webpages.</p>

☆Line break

8.The <br> tag causes the next element or text to display on a new line.

<p>This is the first line.<br>
This is the second line.</p>

☆Subscript Text☆

9.The<S> tag is used to display subscript text,typically appearing slightly below the baseline of the surrounding text.

<p>The chemical formula for carbon dioxide is Co<sub>2</sub></p>

☆Unordered List☆

    ✦✦CSS Guide✦✦

    CSS(cascading style sheets) is a style sheet language which is used to design your webpage.

    There are 3 primary method for applying CSS to HTML Documents :Inline,External and Internal styles.

    ☆Color☆

    The color property sets the color of text content.

    <p>This Text is blue.</p>

    p{color:Blue;}

    ☆Font-Family☆

    The font-family sets the font family for text content.

    <p>This text uses Franklin Gothic Medium font.</p> 
    p{font-family:'Franklin Gothic Medium';}

    ☆Font Size☆

    The font-size property sets the size of the text

    The font size of this text is 10px

    p{font-size:10px}

    ☆Font weight☆

    Font-weight sets the thickness of the font.

    The font weight of this text is 800.

    p{font-weight:800;}

    ☆Margin☆

    The margin property sets the margin space around an element.

    This paragraph has a margin of 50pixels

    p{margin:50px;}

    ☆Padding☆

    The padding property sets the padding space inside the element.

    This paragraph has a padding of 20pixels 
    p{padding:20px;}

    ☆Background color☆

    The Background-color property sets the background color of an element.

    This paragraph has a background color of #FFE4CF. 
    p{background-color:#FFE4CF;}

    ☆Text Align☆

    The Text-align property sets the horizontal alignment of text content.

    This paragraph has centered text alignment. 
    p{text-align:center;}
